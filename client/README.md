## 개요

- 밸런스 게임, 투표 등의 커뮤니티 기능을 제공하는 웹 사이트입니다.

## 목적

- 개발 프로젝트 팀 빌딩시 밸런스게임을 통한 아이스브레이킹과 투표를 통한 임원선출을 지원하기 위함입니다.

## 기술

- React
- Redux
- Styled-Components
- Firebase
